BurpFuzz Plugins for BurpSuite

La idea de crear este plugins con mi amigo @secenv, surgio por la necesidad de reducir el tiempo en la creación de template para buscar vulnerabilidades. Con esta idea podriamos crear muchisimos templates por cada solicitud que haciamos a un dispositivo "Router".

Ahora esto se puede adaptar a muchos otros casos y aqui es donde entrar la imaginacion de todos. Ya que podemos mejorar este plugins llevando esta idea a otras herramientas mas poderosas y compartir con la comunidad.

Description:
1- Para agregar este plugins a burpsuite solo necesitan importar el archivo "BurpFuzz.py" en "Add" de tipo python.



2- Configurar el directorio que almacenara los templates creados.


3- Cuando se obtien un request, podemos crear un template seleccionando la opción "Create BooFuzz Template" desde menu.


4- Se viualiza en nombre del template por ej: "boofuzz_1.py". Este identificador cabia según la cantidad de template que se han creado.


5- Verificando el contenido del template creado.


6- Ejecutamos el template.
